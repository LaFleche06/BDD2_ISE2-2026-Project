# Configuration de la base de données

L'application utilise **SQL Server** avec **SQLAlchemy** et le driver **pyodbc**.

La connexion à la base est définie dans une variable d'environnement appelée `DATABASE_URL`.

---

# 1. Fichier `.env`

Créer un fichier `.env` à partir du fichier d'exemple :

```
cp .env.example .env
```

Puis modifier la variable `DATABASE_URL`.

---

# 2. Format général de `DATABASE_URL`

```
mssql+pyodbc://UTILISATEUR:MOTDEPASSE@HOTE:PORT/NOM_BASE?driver=NOM_DRIVER
```

Signification des paramètres :

* **UTILISATEUR** : utilisateur SQL Server
* **MOTDEPASSE** : mot de passe de l'utilisateur
* **HOTE** : adresse du serveur (ex: `localhost`, IP, ou endpoint cloud)
* **PORT** : port SQL Server (1433 par défaut)
* **NOM_BASE** : base de données à utiliser
* **driver** : driver ODBC utilisé pour communiquer avec SQL Server

---

# 3. Le driver ODBC

SQLAlchemy ne parle pas directement à SQL Server.
La communication passe par **ODBC**, via la bibliothèque Python **pyodbc**.

Il faut donc installer un **driver ODBC SQL Server** sur la machine.

Les drivers les plus courants sont :

* `ODBC Driver 18 for SQL Server` (le plus récent)
* `ODBC Driver 17 for SQL Server`

Dans l'URL, les espaces doivent être remplacés par `+`.

Exemple :

```
ODBC Driver 18 for SQL Server
```

devient

```
ODBC+Driver+18+for+SQL+Server
```

---

# 4. Exemple complet

```
DATABASE_URL=mssql+pyodbc://sa:password@localhost:1433/universite_db?driver=ODBC+Driver+18+for+SQL+Server
```

---

# 5. Exemples selon l'environnement

## Base locale

```
DATABASE_URL=mssql+pyodbc://sa:password@localhost:1433/ma_base?driver=ODBC+Driver+18+for+SQL+Server
```

## Serveur distant

```
DATABASE_URL=mssql+pyodbc://user:password@192.168.1.10:1433/ma_base?driver=ODBC+Driver+18+for+SQL+Server
```

## Cloud (ex: AWS RDS)

```
DATABASE_URL=mssql+pyodbc://admin:password@database.xxxxxx.rds.amazonaws.com:1433/ma_base?driver=ODBC+Driver+18+for+SQL+Server
```

---

# 6. Vérifier les drivers installés

Sur certaines machines, plusieurs drivers ODBC peuvent être installés.

Avec Python :

```python
import pyodbc
print(pyodbc.drivers())
```

Cela affiche la liste des drivers disponibles.
Le nom utilisé dans `DATABASE_URL` doit correspondre exactement à l'un de ces drivers.
